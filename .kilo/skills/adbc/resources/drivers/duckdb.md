# duckdb

## Installing the Driver

Install the DuckDB ADBC driver with dbc by running:

```sh
dbc install duckdb
```

## Connecting

Connect by setting either the `uri` or `path` (e.g., `db.duckdb`) option or omit to use an in-memory database.

## More Information

See https://duckdb.org/docs/stable/clients/adbc#python for more options.
