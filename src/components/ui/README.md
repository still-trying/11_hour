# UI Components Layer (`src/components/ui`)

## Responsibility

This directory contains atomic, presentation-only components (such as buttons, cards, inputs, progress bars, drawers, and timers) that construct the visual design system of 11_HOUR.

## Boundary Rules

- **Allowed Imports**: Animation packages (`framer-motion`, `lucide-react`), basic React state.
- **Forbidden Imports**: Core business logic, network states, global context hooks, or database dependencies.
- **Layer Owner**: Presentation Layer.
