# Layout Components Layer (`src/components/layout`)

## Responsibility

This directory houses multi-page structural scaffolding components that manage global page layouts, responsive frame spacing, and headers/footers.

## Boundary Rules

- **Allowed Imports**: Generic UI components (`@/components/ui/*`), state stores (`@/stores/*`), hooks (`@/hooks/*`), and utility functions (`@/lib/*`).
- **Forbidden Imports**: Feature-specific screens (e.g., dashboard, rescue), direct database layers, or AI models.
- **Layer Owner**: Presentation Layer.
